# Localization architecture (foundation)

This repository includes a **minimal localization foundation** intended to support:

- consistent formatting (dates/numbers) via `Intl`
- a clear default locale and supported locale allowlist

It currently supports:

- `en-US` only

No broad multilingual claims are made in this wave.

## Files

- `lib/i18n/types.ts`
- `lib/i18n/config.ts`
- `lib/i18n/formatting.ts`
