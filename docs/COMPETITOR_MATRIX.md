# Competitor matrix (public)

## What the scores mean

The competitor matrix is a **public-site positioning tool**, not an objective universal benchmark.

**Scores represent current public-site competitive pressure** in the category LeadIntel is pursuing:

- signal-based outbound
- why-now intelligence
- send-ready outreach

Scores reflect:

- public positioning clarity
- workflow depth
- proof and inspectability
- trust center maturity
- actionability
