# Trust Center

## Goal

The Trust Center makes it easy for buyers to verify what exists **today** without relying on sales-driven claims.

## Contents

The Trust Center index (`/trust`) links to:

- Security (`/security`)
- Privacy (`/privacy`)
- Terms (`/terms`)
- Acceptable Use (`/acceptable-use`)
- Subprocessors (`/subprocessors`)
- DPA (`/dpa`)
- Status (`/status`)
- Version (`/version`)
- Changelog (`/changelog`)
- Roadmap (`/roadmap`)

## Truthfulness rules

Do not claim:

- SOC 2 / ISO 27001
- SSO / SAML / SCIM
- “enterprise-ready” features that are not implemented

If something is planned, it belongs in `/roadmap` as **directional focus**, not as a committed ship date.
