# Expansion intelligence (bounded)

In this repo, LeadIntel’s “expansion watch” is a **workflow flag and signal posture**, not a claim of installed-base or upsell certainty.

## What it uses

- explicit program state (`expansion_watch`)
- observed intent and momentum posture (when available)
- follow-through activity and outcomes (when recorded)

## What it does not use / claim

- no product usage telemetry
- no customer success platform data
- no whitespace prediction engine
- no hierarchy certainty

## UI and workflow

- Mark an account as `expansion_watch` via the coverage program state.
- Use planning + next-best-action guidance to run a bounded expansion motion and record outcomes.
