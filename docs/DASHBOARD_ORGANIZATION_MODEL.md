# Dashboard organization model

## Target hierarchy
1. **Page title + concise subtitle**
2. **Plan / preview status + upgrade summary** (supporting, not dominating)
3. **Key metrics**
4. **Primary workflow**
   - Next best step
   - Generate (pitch/report)
5. **Secondary modules**
   - Activation/progress
   - Recent activity
6. **Tertiary**
   - Signals/events
   - Team/ops (only when eligible)

## Starter (preview) emphasis
- Focus on **getting value now** (generate preview, learn the workflow).
- Advanced surfaces are either:
  - hidden, or
  - locked with calm explanation,
  - never “broken”.
