# Forecast support (directional)

LeadIntel provides **directional forecast support** for planning discussions.

## What it is

- Buckets and cautions derived from observed workflow activity
- Useful for weekly planning and “where to invest follow-through”

## What it is not

- not a forecasting platform
- not close prediction
- not revenue numbers
- not pipeline guarantees

## API

- `GET /api/team/forecast-support`

## UI

- `/dashboard/planning` includes a “Forecast support (directional)” section.
